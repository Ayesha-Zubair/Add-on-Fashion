# Add-On-Fashion
Add-On-Fashion is an e-commerce application built using React and firebase.
### Website Link : [Add-On-Fashion](https://add-on-fashion.netlify.app/)
### Work In Progress
More features is being added weekly, as more work needs to be done.
   
#### Current functionality includes:

✔ Authentication with Google Sign In

✔ Authentication with Gmail and Password

✔ Responsive Design

✔ Add items to cart

✔ Remove/delete items from cart

✔ Load/Update data using Firebase
